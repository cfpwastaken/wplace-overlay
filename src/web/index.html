FAQ

How is this different from Blue Marble?

Blue Marble is a local overlay, meaning each user has to manually add and maintain the artworks they want to build.
Wplace Overlay is a global overlay, meaning everyone using it sees the same artworks, exactly as intended by the project owners.

How can I add my projects or alliance?

Contact us on Discord and we will guide you through the process.
For alliances, we require at least 10 members and a proof that you are an admin of the alliance (e.g. a screenshot of the alliance settings).
